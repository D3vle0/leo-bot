# 리오봇
Devleo의 첫번째 디스코드 봇!
(node.js, discord.js)

---
## 명령어
리오야 도움 - 간단 도움말 표시 (느낌표 없음)  
!hello - 당신에게 인사를 합니다.  
!한강 - 현재 한강 수온 표시  
!날씨, !기온 - 현재 날씨정보 표시  
!비트코인, !가상화폐 - 인기 가상화폐 4종의 시세 확인  
!일출, !일몰 - 오늘 일출, 일몰 시간 표시  
!국밥 - 입력한 금액이 몇 국밥인지 계산  
!웹페이지 - 개발자의 웹페이지  
!개발자 - 개발자 정보 표시  
!업데이트 - 나중에 추가할 사항 표시

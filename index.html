봇 소개

디스코드 서버를 더 재밌고 편리하게 이용할 수 있는 리오봇!

주요 기능

  • Open API를 활용

    여러 Open API 들을 사용하여 현재 날씨, 기온, 가상화폐 시세, 일출/일몰 시간 등을 확인할 수 있습니다.

  • 체계적인 코드

    개발자 Devleo의 효율적인 199줄 코드로 언제나 가벼운 성능으로 최상의 품질을 유지할 수 있습니다.

  • 인사 기능

    친구가 없는 당신을 위해 리오봇이 인사해줍니다!

명령어

리오봇의 명령어들을 알아보아요!

  • 199 줄의 코드
  • 1 개의 활성 서버
  • 10 명의 사용자
  • 7 일의 개발기간
  • 0 윤태 수학 점수 
리오야 도움 - 간단 도움말 표시 (느낌표 없음)
!hello - 당신에게 인사를 합니다.
!한강 - 현재 한강 수온 표시
!비트코인, !가상화폐 - 대표 가상화폐 4종 현재 시세 표시
!일출, !일몰 - 일출, 일몰 시간 표시
!날씨, !기온 - 현재 기온과 날씨 표시
!웹페이지 - 개발자의 웹페이지
!개발자 - 개발자 정보 표시
!업데이트 - 나중에 추가할 사항 표시
